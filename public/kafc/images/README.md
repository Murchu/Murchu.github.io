# Public images

Files here are copied to the finished site exactly as supplied. They receive no
Astro image optimisation.

Use this folder only when an asset needs a permanent public URL:

- `placeholders/` — fixed fallback graphics used when CMS media is missing.
- `social/` — default Open Graph and social-sharing images.

Most club branding, sponsor logos, and permanent site photography belongs in
`src/assets/images` instead.

Do not put WordPress news uploads here.
